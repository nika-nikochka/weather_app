## 🌤 Погодный информатор

Учебный проект по разработке десктопного приложения с графическим интерфейсом для получения актуальной информации о погоде, прогноза на несколько дней и анализа исторических данных. Приложение использует бесплатное API Open-Meteo и не требует регистрации или API-ключей.
